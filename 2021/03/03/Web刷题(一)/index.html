[SUCTF 2019]Pythonginx

考点：2019black hat一个议题

python 中 urlparse 模块介绍

简单来看就是得绕过前两个if，进入第三个if才能读取url内容，第一第二的条件一致，url中不能出现suctf.cc，而第三个if要url中有suctf.cc

方法一：查找特殊字符替代寻常字符

在unicode中字符℀(U+2100)，当IDNA处理此字符时，会将℀变成a/c，因此当你访问此url时，dns服务器会自动将url重定向到另一个网站。如果服务器引用前端url时，只对域名做了限制，那么通过这种方法，我们就可以轻松绕过服务器对域名的限制了。

寻找可以代替suctf.cc中随便一个字符即可，如ⓤ ，在服务器中会解析为u，那么就可构造sⓤ ctf.cc
