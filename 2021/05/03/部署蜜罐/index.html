部署Hfish

发表于2021-05-03|更新于2021-05-03
|字数总计:329|阅读时长:1分钟|阅读量:

选择Hfish蜜罐安装地址

不过这是基于go语言开发的，所以以防万一，也顺便记录如何配置go环境

可以直接选择命令行安装

apt install golang

也可以选择手动安装

首先获取go语言的安装包，可以选择下载到本地再传到服务器上，也可以直接在服务器上运行wget下载

完成之后解压到/usr/local

编辑环境变量

在环境变量最后添加GOROOT环境变量，GOROOT变量为go的安装目录，类似java的jdk安装目录，GOPATH类似eclipse的workspace

刷新环境变量

最后检查是否安装成功，安装成功后go version会显示go的版本

同理将蜜罐的压缩包下载好

在当前目录创建一个路径解压安装包

将安装文件包解压到hfish目录下

防火墙开启4433端口

由于我连防火墙都没开，所以开启端口不存在，那么直接就可开启服务
